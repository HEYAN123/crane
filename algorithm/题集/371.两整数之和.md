# 两整数之和

## 题目

不使用运算符 + 和 - ​​​​​​​，计算两整数 ​​​​​​​a 、b ​​​​​​​之和。

示例 1:

输入: a = 1, b = 2

输出: 3

示例 2:

输入: a = -2, b = 3

输出: 1

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/sum-of-two-integers

---

## 题解

- 两个二进制数异或得到无进位相加结果
- 两个二进制数相与得到相加时候的进位
- 所以两数相加 = 异或结果 + 相与结果

```javascript
/**
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
var getSum = function(a, b) {
    return b === 0 ? a : getSum(a^b, (a&b)<<1);
};
```
